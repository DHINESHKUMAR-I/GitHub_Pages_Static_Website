# GitHub Pages Static Website

## Objective

Deploy a colorful static HTML website using GitHub Pages.

## Technologies Used

- HTML5
- CSS3
- JavaScript
- GitHub Pages

## Features

- Responsive Design
- Animated Background
- Navigation Bar
- Hero Section
- Feature Cards
- Contact Section
- Smooth Scrolling
- Typing Animation
- Hover Effects

## Deployment Steps

1. Created index.html
2. Pushed project to GitHub
3. Enabled GitHub Pages
4. Selected main branch
5. Accessed the generated live website
